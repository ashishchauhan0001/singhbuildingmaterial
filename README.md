# Singh Building Material

## Project Overview
**Singh Building Material** is a web application designed for an online building material store. 


## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Next.js

## Installation & Setup
### Prerequisites
Ensure you have the following installed on your system:
- Node.js
- Git

### Steps to Run Locally
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/singh-building-material.git
   ```
2. Navigate to the project folder:
   ```sh
   cd singh-building-material
   ```
3. Install dependencies:
   ```sh
   npm install
   ```

4. Navigate to the frontend folder and install dependencies:
   ```sh
   cd frontend
   npm install
   ```
5. Start the frontend server:
   ```sh
   npm start
   ```
6. Open `http://localhost:3000` in your browser.
